# 🛤️ 중등 수학 디지털 탐구 도구함 (정수와 유리수)

와우중학교 수학 수업에서 학생들의 직관적인 개념 이해를 돕기 위해 제작된 시뮬레이터 통합 페이지입니다.

## 🌟 주요 기능

본 프로젝트는 중학교 1학년 '정수와 유리수' 단원의 핵심 원리를 시각적으로 탐구할 수 있는 두 가지 도구를 제공합니다.

1.  **🔘 셈돌 상쇄 시뮬레이터 (덧셈과 뺄셈)**
    * 양의 셈돌(+)과 음의 셈돌(-)을 직접 움직여 상쇄되는 과정을 실험합니다.
    * 뺄셈을 '더하는 수의 부호를 바꾸어 더하는' 원리로 변환하는 과정을 시각화합니다.
2.  **Tracks 꼬마 기차 레일 시뮬레이터 (곱셈)**
    * 기차의 속도(양수/음수)와 시간(전/후)의 곱을 통해 정수의 곱셈 원리를 이해합니다.
    * 수직선 위에서 기차의 위치 변화를 실시간으로 추적합니다.
